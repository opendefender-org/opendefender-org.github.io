Moduler i SnapThreatMap-økosystemet

Demo- og kunnskapsmoduler – ingen persondata

Denne siden samler fagmoduler som bygger opp under SnapThreatMap. Noen er rene demovisninger, andre er kunnskapsmoduler som kan brukes i dialog med skole, SLT, politi og forebyggende tjenester. Alle eksempler er syntetiske eller sterkt abstraherte, og kan videreutvikles lokalt innenfor gjeldende lovverk.

Viktig: Modulene under er laget for faglig støtte, ikke operative beslutninger. All bruk av virkelige data må forankres lokalt med egne vurderinger av personvern, juss og mandat. SnapThreatMap er en åpen, etisk plattform som skal hjelpe ansvarlige aktører å se mønstre uten å peke ut enkeltpersoner.
Pilot · syntetiske varsler

Pilotdashboard – varsler og risikosoner (demo)

Viser hvordan en kommune, skole eller SLT-gruppe kan få oversikt over varsler, risikosoner og trender uten å se enkeltbrukere. Alle tall og kartsoner er kunstig generert.

  • • Illustrerer anonymisert kartvisning og tidsutvikling.
  • • Egnet som grunnlag for faglig drøfting i pilotkommuner.
  • • Kan kobles mot lokale data senere, under lokal kontroll.

Fagmodul · spillplattformer

Risikomatrise – spillplattformer

Kunnskapsmodul som synliggjør hvordan spillplattformer kan utnyttes til rekruttering, grooming og påvirkning – og hvilke beskyttende faktorer som kan bygges rundt barn og unge.

  • • Strukturert oversikt over risiko- og beskyttelsesfaktorer.
  • • Kan brukes i dialog med foresatte, skole og utekontakt.
  • • Underlag for lokale handlingsplaner og undervisningsopplegg.

Demo · fiktive låter

Risikomatrise – musikkinnhold (fiktiv demo)

Utforsker hvordan innhold i låter kan kategoriseres etter tematikk, for eksempel glorifisering av vold og kriminell livsstil, uten å knytte vurderingen til faktiske artister eller profiler.

  • • Bruker kun fiktive artister og låter.
  • • Fokus på budskap og narrativ – ikke på enkeltpersoner.
  • • Kan inngå i samtaler om balanse og motfortellinger i ungdomskultur.

Planlagt · kunnskapsgraf

Multiplattformanalyse og kunnskapsgrafer (plan)

Planlagt modul som skal koble mønstre på tvers av plattformer (for eksempel spill, sosiale medier og åpne musikkdata) i en kunnskapsgraf. Målet er å gi ansvarlige aktører bedre oversikt over risikobilder, uten å lagre mer persondata enn nødvendig.

  • • Utforsker bruk av kunnskapsgrafer for forebygging.
  • • Bygger på aggregerte og anonymiserte mønstre, ikke enkeltprofiler.
  • • Krever tydelige rammer for etikk, juss og lokal forankring.

Fagmodul · kunnskapsgraf

Sociograf – gaming og kriminalitetsnarrativer

Modul som analyserer åpne samtaler om «crime RP» og lignende digitale miljøer på tvers av plattformer (f.eks. Reddit, Discord og TikTok), og visualiserer dem som en kunnskapsgraf over narrativer, risiko og forebyggende tiltak.

  • • Kartlegger hvordan ungdom ledes mellom plattformer og RP-miljøer.
  • • Viser kriminalitetsnarrativer som oppstår i gaming og sosiale medier.
  • • Knytter hvert narrativ til konkrete tiltak for skole, SLT og foresatte.

Planlagt

Kommunedata og SSB/Politi-statistikk (plan)

Modul som samler åpne statistiske data (for eksempel SSB og offentlige årsrapporter) på kommunenivå, slik at lokale aktører kan se forebyggingsarbeid og utvikling over tid i samme bilde.

  • • Fokus på åpne, aggregerte tall – ikke individdata.
  • • Kan sammenlignes på tvers av kommuner og regioner.
  • • Underlag for politiske beslutninger og ressursprioritering.

Planlagt

DIPD – Desentralisert stegvis påvirkning (plan)

Videreføring av DIPD-konseptet der trinnvis digital påvirkning og rekruttering modelleres som mønstre over tid, på tvers av kanaler. Skal primært brukes til faglig forståelse og risikovurdering.

  • • Vekt på mønstre, ikke enkelthendelser.
  • • Kan støtte SLT og politi i strategisk forebygging.
  • • Krever tydelig metodikk og robust etikkramme.
