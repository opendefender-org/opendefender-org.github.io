# opendefender-org.github.io

OpenDefender er en nøytral plattform for åpne verktøy og forskning innen digital trygghet,
OSINT, forebygging og cybersikkerhet – utviklet for skoler, kommuner og sikkerhetsmiljøer.

Dette repoet inneholder organisasjonens offentlige nettside.

## SnapThreatMap (prototype)

SnapThreatMap er første modul i OpenDefender-initiativet. Målet er å gi ansvarlige aktører
(etat, skole, kommune, forebyggende politi) bedre oversikt over digitale risikomønstre,
uten å samle inn eller lagre data om enkeltpersoner.

- **Forside:**  
  https://opendefender-org.github.io/

- **Trusselkart (demo):**  
  https://opendefender-org.github.io/kart.html

- **Varsle anonymt (demo):**  
  https://opendefender-org.github.io/varsle.html

> Merk: Kart- og varslingssider er per nå kun demonstrasjoner. All bruk mot reelle data
> skal skje under tydelig mandat, lokal datakontroll og dokumenterte etiske rammer.
